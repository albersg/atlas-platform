# Indice de deployment

Esta pagina agrupa los runbooks de despliegue y promocion que soportan el modelo
operativo actual del repositorio.

- `k3s/RUNBOOK.md`: runbook no productivo para `dev` y `staging` sobre k3s.
- `gitops/ARGOCD_SOPS_RUNBOOK.md`: bootstrap y operacion de Argo CD + KSOPS + SOPS.
- `releases/IMAGE_PROMOTION.md`: publicacion inmutable y promocion de `staging` por digest.

Entrada recomendada para una vista mas ejecutiva: `docs/operations/overview.md`.
